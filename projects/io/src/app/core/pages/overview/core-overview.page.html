Core

@ngez/core is a library for Angular applications, which consists mostly of directives and other utilities, although our few components can still easily be overridden with new styles. This way you're not constrained by the theme of a component framework. Instead you can focus on how the view should look with your own styles, and we'll take care of the functionality.

Tested on Chrome, Firefox and IE11.
