File Dropzone

Turn any element into a file dropzone and you decide how you want to render the files. The control's value will be the File[].

Features

  • Works with FormsModule and ReactiveFormsModule.
  • Dropping a folder will extract the files.
  • Easily style the dropzone while dropping files.

Caveat

Internet Explorer doesn't allow dropping folders or extracting the files from a folder, so if the dropped files contain at least one folder, it'll return an empty array. Works as expected on Edge.
